# 项目编码规范

1. 所有.py文件头部必须包含 # -*- coding: utf-8 -*-
2. 所有文件读写操作显式指定 encoding='utf-8'
3. 绝对禁止在代码中硬编码任何密钥、密码、API Key、数据库连接字符串、文件路径
4. 所有环境相关配置通过 config_loader.get_env() 获取
5. 所有可调参数通过 config_loader.get_config() 获取
6. 所有游戏数据参数（角色面板、技能倍率、敌人属性等）存放在独立的数据文件中
7. 所有输出文件以UTF-8编码保存
8. 所有对外输出文本内容在生成时显式使用UTF-8编码
